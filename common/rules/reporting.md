# Reporting rules (all subjects)

- Хранить исходники, артефакты запуска и итоговый отчет в одном предметном каталоге.
- Все графики должны быть читаемыми: подписи осей, единицы измерения (если есть), легенда, заголовок.
- Скрипты должны запускаться воспроизводимо одной командой.
- При наличии требований ВАК/ГОСТ использовать шаблоны из `common/templates/`.
