# OMM workflow rules

1. Сначала фиксировать математическую постановку и граничные/начальные условия.
2. Выбирать численный метод на основе методички из `subjects/omm/resources/`.
3. Реализовывать расчетный код (Go по умолчанию, Python при явной выгоде для численных методов/графиков).
4. Проверять сходимость на серии сеток.
5. Формировать отчет с графиками и кратким анализом устойчивости/погрешности.
